# WindexTech — Static Site

Static HTML/CSS/JS marketing site. No build step.

## Deploy (Vercel)
Vercel auto-detects this as a static site. No framework, no build command, output is the repo root.

`vercel.json` enables clean URLs (`/contact` instead of `/contact.html`).

## Pages
- index.html, about.html, services.html, case-studies.html, contact.html, ai-readiness-audit.html
